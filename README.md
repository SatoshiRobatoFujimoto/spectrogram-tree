# spectrogram-tree
すばやくスペクトログラムが見たいという状況によく出くわすので、さくっと確認するために作ったツール。  

## このアプリはなに？  
ディレクトリツリーのアイテムをダブルクリックするだけでスペクトログラムが見れます。
signalとスペクトログラムを並べつつ、音も再生できます。
